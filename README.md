# iPhone Page

## Description

A web application where I tried to replicate the signature Look & Feel of Apple pages and animations.

## Tech Stack

- React
- Tailwind
- GSAP
- Three.js
- Sentry

## Highlights

- Interactive iPhone 3D-model
- Animation Scroll Triggers
- Performance & Error Tracking
